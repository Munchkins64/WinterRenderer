# WintrRenderer
Its a complete renderer with FSR 2.0.

Features:
- PBR
- FieldityFX Super Resolution
- GPU accleration
- Path tracing

Many of these features are yet to be designed..

> note: i'm still learning
<hr>

## when cloning
This renderer repo contains a `.gitmodules` file so, its better to run this command: <br>
`git clone --recurse-submodules https://github.com/Munchkins64/WinterRenderer`
